# 7-Eleven-Database-Foundations-Project

A PostgreSQL database designed to support the daily operations of a multi-branch 7-Eleven convenience store. The system manages core business processes including product and inventory management, supplier deliveries, sales transactions, customer loyalty, store expenses, and employee shift scheduling.


## Project Overview

- This project models the database system of a multi-branch 7-Eleven convenience store. It is designed to centralize and organize operational data across different store locations while maintaining data consistency and integrity.

- The database supports the core business operations of the store, including product catalog management, branch inventory tracking, supplier deliveries, point-of-sale transactions, All Member loyalty records, store expense management, and workforce scheduling.

- The objective of this project is to design a normalized relational database that accurately represents the business requirements of a modern convenience store while providing efficient SQL functions, procedures, triggers, views, and indexes to support common operational and reporting tasks.
